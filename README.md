# Online Auction System

A web-based platform for real-time bidding on products.

## Features
- User login & registration
- Buyer & Seller roles
- Auction creation
- Real-time bidding
- Countdown timer
- Winner declaration
- Admin dashboard
- Product image upload

## Tech Stack
- Frontend: React.js
- Backend: Node.js / Spring Boot
- Database: MySQL

## How to Run
1. Start backend
2. Start frontend
3. Open browser at localhost
